# JS tests
This is a Repository with different variations of js code testing

## Run Locally

Clone the repository

```bash
  git clone 
```

Go to one of the project folders

```bash
  cd (folder name)
```

Install dependencies

```bash
  npm install
```

Start the testing

```bash
  npm run test
```
